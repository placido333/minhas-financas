# Minhas Finanças

App pessoal de finanças (1 página, instalável no celular). Os dados ficam só no seu aparelho.

## Atualizar o app
Abra esta pasta no Claude Code e diga o que quer mudar. Ele edita, dá `git push`
e o GitHub Pages republica sozinho no mesmo link.

## Como registrar gastos
Digite `Categoria - valor` com o sufixo da moeda:
- `Ifood - 101,50R` → desconta do Livre em Real
- `Uber 25D` → desconta do Livre em Dólar
- sem letra no fim = Real

O olhinho no topo borra/revela os valores.
